About the Workshop

Python has been the most-used programming language by GIS experts and researchers due to its simplicity, massive user base, powerful and robust external libraries. This workshop focuses on how you can use Python to process and manipulate geospatial data using popular libraries such as GDAL, RasterIO and Geopandas. After the completion of this workshop, you will be able to break free from using the conventional GIS tools such as ArcGIS and QGIS etc. and you will be able to write/make tools of your own.

The course contents has been designed by keeping beginner Python users in mind. However, beginner to intermediate level of knowledge in geospatial sciences will help the participants to understand the concepts better.
